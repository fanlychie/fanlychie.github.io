参数 类型 描述
healthy 布尔 文件上传报告是否健康，若是，表明文件上传全部成功，否则表示存在文件上传失败或空
failedNumber 整型 文件上传失败的个数
successfulNumber 整型 文件上传成功的个数
fileKeys 字符串数组 上传成功的文件 Key 列表，每个 Key 表示本地文件的地址
failedMsgs 字符串数组 文件上传失败的信息列表
